# 📊 360° Banking Analytics Dashboard

## 📌 Project Overview

This project presents an interactive **360° Banking Analytics Dashboard** built using Tableau. The dashboard provides a 360° view of banking operations, including customer accounts, transactions, loan portfolio, credit card usage, and customer feedback.

It is designed to help business stakeholders quickly monitor key performance indicators, identify trends, and make data-driven decisions.

---

## 🎯 Objectives

- Analyze customer financial behavior using segmentation
- Evaluate loan approval performance and risk
- Monitor credit card utilization and customer spending patterns
- Track customer feedback and service quality
- Provide a centralized executive view of banking KPIs

---

## 📊 Dashboard Features

- ✅ KPI Cards:
  - Total Accounts
  - Total Account Balance (Millions)
  - Loan Approval Rate (%)
  - Card Utilization (%)
  - Feedback Resolution Rate (%)

- ✅ Customer Insights:
  - Age Group segmentation
  - City-wise distribution
  - Account type analysis

- ✅ Financial Analysis:
  - Transaction trends over time
  - Account balance distribution
  - Loan portfolio (Auto, Mortgage, Personal)

- ✅ Credit Card Analytics:
  - Card type distribution (Visa, AMEX, MasterCard)
  - Credit utilization behaviour
  - Risk identification

- ✅ Customer Experience:
  - Feedback type analysis (Complaint, Suggestion, Praise)
  - Resolution tracking

---

## 📈 Key Metrics

- **Total Accounts** → COUNTD(Customer ID)
- **Total Balance** → SUM(Account Balance)
- **Loan Approval Rate** → Approved / Total Loans
- **Card Utilization** → Balance / Credit Limit
- **Feedback Resolution %** → Resolved / Total Feedback

---

## 🛠 Tools & Technologies

- Tableau (Dashboard & Visualization)
- Excel (Data Preparation)
- Calculated Fields (KPIs & Metrics)

---

## 💡 Key Insights

- Mid-career customers (35–44) hold the highest account balances  
- Younger customers show higher transaction activity but lower balances  
- Loan approval rate reflects credit performance efficiency  
- High credit card utilization segments indicate potential financial risk  
- Feedback analysis helps identify service improvement areas
